Сообщение в будущее: не забыть создать виртуальное окружение с необходимыми библиотеками
 перед запуском программы
 В методе playsound заменить путь файла на актульный. Слеш прямой должен быть  -"/"
 Должен быть указан полный путь к файлу myiu.mp3 (тогда котик миукает), либо к roar.mp3 (тогда рычит)
